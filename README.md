# boAt Lifestyle - Website Project

A modern, responsive website for boAt Lifestyle's wireless headphones and audio products.

## Project Overview

This is a multi-page website showcasing boAt's premium audio products, including detailed information about the brand, product offerings, services, and pricing. The site features a clean, modern design with an emphasis on user experience and product highlighting.

## Project Structure

```
project-root/
├── index.html          # Home page - Product showcase
├── about.html          # About us - Brand story and mission
├── services.html       # Services page
├── pricing.html        # Pricing information
├── contact.html        # Contact form and information
├── main.css            # Main styling
├── layout.css          # Layout and structure styles
├── animations.css      # Animation effects
└── README.md           # Project documentation
```

## Pages

- **Home** - Product showcase featuring boAt's wireless headphones with pricing and key features
- **About** - Company history, mission, and brand story
- **Services** - Available services and support
- **Pricing** - Product pricing and plans
- **Contact** - Contact information and communication options

## Design Features

- **Responsive Navigation** - Easy-to-navigate menu across all pages
- **Product Cards** - Container-based layout for product display
- **Modern Styling** - Clean, professional design using blue (#2563eb) accent color
- **Animations** - CSS animations for enhanced user experience
- **Mobile Ready** - Viewport meta tags for responsive design

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling and layout
  - layout.css - Page structure and layout
  - main.css - Component styling
  - animations.css - Dynamic effects
- **JavaScript Ready** - Infrastructure for future interactivity

## Getting Started

1. Open `index.html` in a web browser
2. Navigate through the site using the top navigation menu
3. View products, company information, and contact details

## Key Features
